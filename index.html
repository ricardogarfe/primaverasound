Primavera Sound 2013 Horarios

Introduction

Retrieve Primavera Sound 2013 schedule from Programación

Environment

Python 2.7.2

Install

Developer requirements

Libraries required:

* requests
* lxml

Install using python dependency management easy_install.

$ easy_install requests
$ easy_install lxml

Command

run the script using:

$ python retrieve_schedule.py

Or import module Schedule in your project and create the instance:

primaveraSoundSchedule = Schedule()

import pprint
pprint.pprint(primaveraSoundSchedule.horarios)
